## 🏁 참여방법

1. 이 저장소를 fork 합니다.
2. fork 한 레포지토리의 HTTPS URL을 복사해 clone합니다. (git clone <URL>)
3. 작업을 한뒤 커밋을 하고 푸쉬합니다.
    - git add
    - git commit -m "[커밋메시지]"
    - git push origin
4. fork한 레포지토리에 pull request 요청
  
  
## 🏁 commit 규칙

-   커밋메시지(코테): "add : leetCode/문제번호/적용자료구조이름/난이도"
-   커밋메시지(개념) : "add : 개념명"
